## Шаг 4. Вывод данных в PHP

Использование языковых конструкций `echo` и `print` для вывода текста. Добавлен тег `<br />` для наглядности (переноса строк) при отображении в браузере.

**Код:**

    <?php
    echo "Hello, World with echo!<br />";

## Шаг 5. Работа с переменными и выводом

Создание целочисленной и строковой переменных, а также их вывод на экран различными методами.

**Код:**

    <?php
    $days = 288;
    $message = "Все возвращаются на работу!";

    // 1. Вывод с использованием конкатенации (оператор ".")
    echo "Прошло дней: " . $days . ". " . $message . "<br />";

    // 2. Вывод с использованием двойных кавычек (интерполяция переменных)
    echo "Прошло дней: $days. $message<br />";

## Контрольные вопросы

### 1. Какие способы установки PHP существуют?

- **Локальные серверные сборки:** Использование готовых пакетов, включающих PHP, веб-сервер (Apache/Nginx) и базу данных (MySQL/MariaDB). Примеры: _XAMPP, WAMP, MAMP, OpenServer_.
- **Пакетные менеджеры ОС:** Установка через командную строку (например, `apt` в Ubuntu/Debian, `brew` в macOS).
- **Ручная установка:** Загрузка бинарных файлов напрямую с официального сайта php.net и ручная настройка путей среды (PATH).
- **Контейнеризация:** Использование _Docker_ для запуска PHP в изолированном контейнере.

### 2. Как проверить, что PHP установлен и работает?

- **Через терминал (командную строку):** Ввести команду `php -v`. Консоль выдаст текущую версию интерпретатора.
- **Через веб-сервер:** Создать файл с кодом `<?php phpinfo(); ?>`, запустить веб-сервер и открыть этот файл в браузере. Появится таблица с конфигурацией PHP.

### 3. Чем отличается оператор echo от print?

- **Возвращаемое значение:** `print` всегда возвращает значение `1` после успешного выполнения. `echo` ничего не возвращает.
- **Параметры:** `echo` может принимать несколько аргументов через запятую. `print` принимает только один аргумент.
- **Скорость:** `echo` работает незначительно быстрее, так как не тратит ресурсы на возврат значения.
