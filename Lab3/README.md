## 1. Условные конструкции

```php
<?php
$dayOfWeek = date('N');

// John Styles
if (in_array($dayOfWeek, [1, 3, 5])) {
    $johnSchedule = "8:00-12:00";
} else {
    $johnSchedule = "Нерабочий день";
}

// Jane Doe
if (in_array($dayOfWeek, [2, 4, 6])) {
    $janeSchedule = "12:00-16:00";
} else {
    $janeSchedule = "Нерабочий день";
}

echo "<b>№ | Фамилия Имя   | График работы </b><br>";
echo "1 | John Styles   | $johnSchedule <br>";
echo "2 | Jane Doe      | $janeSchedule <br>";
?>
```

## 2. Циклы

```php
<?php
echo "<h3>1. Цикл FOR с выводом промежуточных значений</h3>";
$a = 0;
$b = 0;

for ($i = 0; $i <= 5; $i++) {
    $a += 10;
    $b += 5;
    echo "Шаг $i: a = $a, b = $b <br>";
}
echo "<b>End of the loop: a = $a, b = $b</b><br><br>";
?>

<?php
echo "<h3>2. Цикл WHILE</h3>";
$a = 0;
$b = 0;
$i = 0;

while ($i <= 5) {
    $a += 10;
    $b += 5;
    echo "Шаг $i: a = $a, b = $b <br>";
    $i++;
}
echo "<b>End of the loop: a = $a, b = $b</b><br><br>";
?>

<?php
echo "<h3>3. Цикл DO-WHILE</h3>";
$a = 0;
$b = 0;
$i = 0;

do {
    $a += 10;
    $b += 5;
    echo "Шаг $i: a = $a, b = $b <br>";
    $i++;
} while ($i <= 5);

echo "<b>End of the loop: a = $a, b = $b</b><br>";
?>
```

## 3. Контрольные вопросы

**1. В чем разница между циклами for, while и do-while? В каких случаях лучше использовать каждый из них?**

- **`for`**: Используется, когда заранее известно точное количество итераций (шагов). Все элементы управления циклом (инициализация, условие, шаг) находятся в одной строке, что делает код компактным.
- **`while`**: Цикл с предусловием. Условие проверяется до выполнения тела цикла. Если оно изначально ложно, цикл не выполнится ни разу. Лучше всего использовать, когда количество итераций заранее неизвестно и зависит от динамических условий.
- **`do-while`**: Цикл с постусловием. Сначала выполняется тело цикла, а только затем проверяется условие. Гарантирует выполнение кода внутри цикла как минимум один раз. Подходит для ситуаций, где необходимо сначала выполнить действие (например, запросить ввод пользователя), а затем решать, повторять ли его.

**2. Как работает тернарный оператор `? :` в PHP?**
Тернарный оператор — это сокращенная форма конструкции `if-else`. Синтаксис: `условие ? выражение_1 : выражение_2;`.
Сначала вычисляется `условие`. Если оно истинно (`true`), оператор возвращает результат `выражение_1` (до двоеточия). Если ложно (`false`) — возвращает результат `выражение_2` (после двоеточия).

**3. Что произойдет, если в do-while поставить условие, которое изначально ложно?**
Тело цикла `do-while` выполнится ровно один раз. Поскольку проверка условия находится в самом конце цикла, интерпретатор выполнит все инструкции внутри блока `do`, после чего проверит ложное условие в `while` и прекратит дальнейшее выполнение цикла.
